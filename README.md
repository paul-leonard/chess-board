## Overview
Two opposing queens exist on the chessboard.  Are they safe?  Only our sophisticated method can know for sure.

## Architecture
This Jupyter notebook running Python code produces a multi-dimensional matrix to render a chess board.  Red and blue queens are added to the board.  A method determines if the queens are under attack from each other.
